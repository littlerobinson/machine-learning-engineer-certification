# jedha_quizz 👽️

## Description ✨

Quizz program in python.

Questions and answers structures :

```python
question_list = [
    {
        "question": "Couleur du cheval blanc d'henry IV ?", 
        "answer": "Blanc"
    },
    {
        "question": "Prefecture des Hautes-Alpes ?", 
        "answer": "Gap"
    },
    {
        "question": "Capitale du Chili ?", 
        "answer": "Santiago"
    },
]
```

## Launch 🚀

```bash
python3 run.py
```
